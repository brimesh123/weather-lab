# Splash Screen Image

## Add Your Splash Image Here!

**Required Image:**
- **Name**: `splash_logo.png`
- **Size**: 1152 x 1152 pixels
- **Format**: PNG with transparency
- **Location**: Place it in this folder

## Quick Steps:

1. Create or design your splash screen image (1152x1152 PNG)
2. Save it as `splash_logo.png`
3. Place it in this `assets/splash/` folder
4. Run: `flutter pub get`
5. Run: `flutter pub run flutter_native_splash:create`
6. Run: `flutter run` to test
